# Coordinator-only replacement audit

This directory is not an annotator input. Its replacement-v3 report includes
evaluator-side identities, oracle diagnostics, and mappings to the rejected
pool. Annotators A/B receive only their own randomized assignment, the matching
blind packet, the annotation template, and the public annotation protocol and
execution guide.

The reports describe candidate construction checks only. They do not constitute
human validation, adjudication, split integration, or sealing.

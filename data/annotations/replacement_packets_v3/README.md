These 63 JSON files are label-free inputs for independent annotators. They
contain only authenticated boundary evidence, public action contracts,
deterministic tool semantics, and scripted user responses.

Do not give annotators the source records under `data/tasks/replacements_v3/`,
the coordinator audit under `research/coordinator_only/`, another annotator's
work, any model output, or the rejected-task mapping. Assignment files and
annotation responses are deliberately not created by the candidate builder.
